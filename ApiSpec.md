# Api Specifications
## Login
**POST /login**

|Key   |Type   |
|:-:|---|
|email   |String   |
|password|String   |

## Sign Up
**POST /signup**

|Key   |Type   |
|:-:|---|
|email   |String   |
|password|String   |
|username|String   |

## Location
**POST /location**

|Key|Type|
|:-:|---|
|lat|Number|
|lng|Number|
|comment (Optional)|String|
